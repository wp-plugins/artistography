=== Artistography ===
Contributors: MistahWrite
Tags: artistography, artist, discography
Requires at least: 3.0
Tested up to: 4.1

Artistography allows you to have a discography on a record label of artists with inter-linked featurings and collaborations among them, or outside artists.

== Description ==


== Installation ==

Install the Artisography plugin to your blog, and Activate it.  Next, Enter an Artist into the Admin/Dashboard > Artistography > Artists, and use the following shortcodes:

[artistography_display_artist_page]
